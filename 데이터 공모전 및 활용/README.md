# 데이터 활용 및 분석: 데이터사이언스 관련 공모전 참여를 통한 공개 데이터의 활용 및 분석 능력


## Overview
This repository presents data-driven projects developed through participation in data competitions and applied analytics tasks. The focus is on leveraging real-world datasets to perform structured data analysis, statistical modeling, and machine learning, while ensuring reproducibility and transparency of the entire workflow.

## Objective
- Apply data analysis and machine learning techniques to competition datasets  
- Develop reproducible pipelines for data preprocessing, analysis, and modeling  
- Extract meaningful insights from large-scale and multi-source data  
- Ensure all results are replicable through shared code and structured workflows  

## Project List
The repository includes the following projects:

### UNDP Hackathon - Data Analysis on Gender and Climate Integration
- Multi-source data integration using OECD CRS, CCVI, ND-GAIN, and UNDP HDR  
- Statistical modeling and spillover analysis for gender–climate aid integration  
- Network analysis and clustering for donor-recipient relationships  

## Reproducibility
All projects are designed to support reproducibility through:
- Structured data preprocessing and integration pipelines  
- Defined feature engineering and analytical methodologies  
- Use of standard data analysis and ML libraries  
- Modular notebooks for end-to-end execution  
- Clear repository structure and documentation  

Each project can be executed independently and includes the necessary steps for data loading, processing, analysis, and visualization.

## Tech Stack
Python, Pandas, NumPy, scikit-learn, Statsmodels, NLP (TF-IDF), Visualization Libraries
