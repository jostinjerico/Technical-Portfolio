# 데이터 수집 및 정제: 데이터 수집 기술과 수집한 데이터 집합의 유용성 및 공개 여부

## Overview
This repository presents projects focused on data collection, structuring, and preprocessing for AI and data-driven applications. It emphasizes transforming raw and domain-specific data into structured, usable formats suitable for analysis, modeling, and intelligent systems.

## Objective
- Collect and organize data from diverse sources  
- Transform raw data into structured and machine-readable formats  
- Develop reproducible data preprocessing pipelines  
- Ensure data usability for downstream AI and analytics tasks  

## Project List
The repository includes the following projects:

### [Manufacturing Chatbot using AI Agent](../데이터%20수집%20및%20정제/Manufacturing%20Chatbot%20using%20AI%20Agent)
- Collected and structured domain knowledge into JSON-based datasets
- Performed data preprocessing and transformation for AI agent retrieval

### [Manufacturing Process Analytics – Preprocessing and EDA](../데이터%20수집%20및%20정제/Manufacturing%20Process%20Analytics%20-%20Preprocessing%20and%20EDA)
- Collected and cleaned manufacturing process data to ensure data quality
- Applied feature engineering and statistical labeling for defect detection

## Reproducibility
All projects are designed to support reproducibility through:

- Structured data collection and preprocessing workflows  
- Explicit feature engineering and transformation steps  
- Use of standard data processing and ML libraries  
- Notebook-based end-to-end execution  
- Clear repository structure and documentation  

Each project can be executed independently and includes the necessary steps for data loading, preprocessing, and analysis.

## Tech Stack
Python, Pandas, NumPy, scikit-learn, JSON, ChromaDB, Visualization Libraries
