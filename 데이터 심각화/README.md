# 데이터 심화: 고급 데이터 분석 및 인사이트 도출

## Overview
This repository presents projects focused on advanced data analysis and insight generation using real-world datasets. The emphasis is on applying exploratory analysis, statistical techniques, and visualization to uncover patterns, trends, and meaningful interpretations from complex data.

## Objective
- Perform in-depth data analysis on real-world datasets  
- Apply statistical and exploratory techniques to extract insights  
- Analyze patterns across temporal, categorical, and spatial dimensions  
- Develop structured and reproducible analytical workflows  

## Project List
The repository includes the following projects:

### Haiti – Data Analysis
- Exploratory analysis of crisis-related datasets  
- Temporal, categorical, and spatial pattern analysis  
- Visualization of humanitarian needs and incident distribution  

### UNDP Dashboard
- Data analysis on development aid datasets  
- Multi-level analysis (donor, sector, recipient)  
- Visualization of funding patterns and integration indicators  

### Manufacturing Process Analytics – EDA and Defect Detection
- Exploratory data analysis of manufacturing process data  
- Statistical defect detection using threshold-based labeling  
- Feature impact analysis and process parameter evaluation  
- Visualization of process trends and defect patterns  

## Reproducibility
All projects are designed to support reproducibility through:

- Structured data analysis and visualization workflows  
- Defined preprocessing and transformation steps  
- Use of standard data analysis libraries  
- Notebook-based end-to-end execution  
- Clear repository structure and documentation  

Each project can be executed independently and includes the necessary steps for data loading, analysis, and visualization.

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Visualization Libraries
